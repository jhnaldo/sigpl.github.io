제목: 손에 잡히는 언어모델 구현

내용

최근 거대 언어 모델을 활용하는 프로그래밍 언어 관련 연구가 늘어나고 있습니다. 이번 튜토리얼에서는 프로그램 소스 코드를 다룰 수 있는 프로그래밍 언어 모델과 다양한 벤치마크 데이터를 소개합니다. 이를 이용해서 직접 모델을 학습하고 실행하는 방법을 실습해보고, 다양한 언어 모델을 효과적으로 이용할 수 있는 방법을 생각해봅니다. 실습에 참여하실 분들은 개인 노트북과 구글 Colab 계정을 미리 준비해주세요.

연사: 류연희 (KAIST)

  • 2018: KAIST PLRG 연구실 석사
  • 2018-2022: Samsung Research SE 팀 연구원. 언어 모델을 이용한 파이썬 소스 코드 분석 및 자동 수정 연구
  • 2022- 현재: KAIST Prosys 연구실 박사 과정, 정적 분석을 이용한 안전한 프로그램 생성 모델 연구
